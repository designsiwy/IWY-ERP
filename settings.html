Quick Setup Steps

  1. Go to script.google.com → New project
  2. Paste Code.gs, API.gs, Sheets.gs from the apps-script/ folder
  3. Run the setup() function once to create the spreadsheet
  4. Deploy → New Deployment → Web app → Execute as: Me → Who has access: Anyone
  5. Copy the Web App URL, paste above, click Save URL
